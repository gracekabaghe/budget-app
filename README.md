

# Budget App

## Getting Started

To get a local copy up and running follow these simple example steps.

- Copy clone of the Repo

```
git@github.com:gracekabaghe/budget-app.git

```

- Go to the repository folder in your command prompt.

```
cd Budget-app
```

```
bundle install
```

```
rails s
```

##Reset DB

```
rake db:drop
rake db:create
rake db:migrate
```
